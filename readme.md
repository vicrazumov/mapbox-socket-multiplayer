# Mapbox Socket.io Multiplayer Prototype

## Usage
0. `npm i`
1. set your Mapbox access token in `public/index.js`
2. `npm run start`
3. open 2 tabs with `localhost:3000` and play!

## Actions supported
1. zooming
2. dragging
3. opening popups
4. closing popups

## NO OPTIMIZATION IMPLEMENTED SO FAR (debouncing, etc)
